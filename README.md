Wave

A local-first semantic memory engine for macOS.

Wave helps you rediscover files, screenshots, PDFs, and documents through AI-powered search and visual graph exploration — fully offline.

⸻

Features

• Semantic search (multilingual E5 model)
• Keyword + recency hybrid ranking
• Visual graph relationships
• Local-only AI processing
• Folder-based indexing
• Search memory learning layer
• Optional integration with Trace

⸻

Philosophy

Wave is built around three principles:
	1.	Local-first
	2.	Privacy by design
	3.	Intelligence without cloud dependency

Your data stays on your machine.

⸻

How It Works

Wave:
	1.	Indexes selected folders
	2.	Extracts metadata and text
	3.	Generates semantic embeddings locally
	4.	Builds relationship graphs
	5.	Ranks results using hybrid scoring

No external APIs are used.

⸻

Technical Stack

• SwiftUI
• CoreML (multilingual-e5-small)
• SQLite
• macOS App Sandbox
• App Groups (optional integration with Trace)

⸻

System Requirements

macOS 14.0 or later
Apple Silicon recommended

⸻

Companion App

Wave can optionally connect to Trace for extended memory exploration.

Both apps remain fully functional independently.
